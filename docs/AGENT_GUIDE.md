# 电商货源猎手智能体 - 用户指南

## 📋 概述

电商货源猎手（E-commerce Sourcing Agent）是一个专业的淘宝/拼多多货源寻找助手，帮助卖家高效寻找潜在热卖商品货源，减少手动调研时间，提升选品效率。

### 核心价值

- **自动化市场洞察**：实时获取市场趋势、热搜关键词、销量排行
- **风险评估**：智能评估供应商质量、知识产权风险、竞争态势
- **供应商匹配**：根据价格、地区、品类等条件筛选优质供应商
- **ROI计算**：精确计算投资回报率和利润空间
- **结构化输出**：提供清晰的推荐列表和行动建议

## 🛠️ 可用工具

### 1. web_search_tool - 基础联网搜索

**功能**：获取实时市场数据、供应商信息和产品趋势

**参数**：
- `query`（必填）：搜索关键词，如"淘宝面膜热卖趋势"、"1688手机壳批发"
- `count`（可选）：返回结果数量，默认10条
- `need_summary`（可选）：是否包含AI生成的摘要，默认True

**返回结果**：
- 搜索结果列表（标题、URL、摘要、来源网站）
- AI生成的总结摘要

**使用场景**：
- 查询市场趋势
- 获取产品信息
- 了解行业动态

---

### 2. advanced_search_tool - 高级联网搜索

**功能**：支持站点过滤和时间范围过滤的高级搜索

**参数**：
- `query`（必填）：搜索关键词
- `search_type`（可选）：搜索类型，支持 "web"（默认）、"web_summary"、"image"
- `count`（可选）：返回结果数量，默认10条
- `sites`（可选）：限定搜索的站点，如"taobao.com,1688.com"
- `time_range`（可选）：时间范围，如"1d"（1天）、"1w"（1周）、"1m"（1个月）

**返回结果**：
- 根据搜索类型返回对应的结果
- AI生成的总结摘要（web_summary类型）

**使用场景**：
- 在特定网站搜索（如仅搜索1688.com）
- 搜索最近一周的热销产品
- 获取带AI总结的搜索结果

---

### 3. image_search_tool - 图像搜索

**功能**：搜索产品图片，用于视觉参考和市场分析

**参数**：
- `query`（必填）：图片搜索关键词，如"面膜产品图"、"手机壳设计"
- `count`（可选）：返回图片数量，默认10张

**返回结果**：
- 图片URL
- 图片尺寸（宽度、高度）
- 来源网站
- 图片标题

**使用场景**：
- 获取产品视觉参考
- 了解产品设计趋势
- 辅助选品决策

---

### 4. roi_calculator_tool - ROI计算器

**功能**：精确计算产品的投资回报率（ROI）和利润率

**参数**：
- `purchase_price`（必填）：进货价（元）
- `selling_price`（必填）：销售价（元）
- `logistics_cost`（可选）：物流费用（元），默认0
- `quantity`（可选）：销售数量，默认1

**返回结果**：
- 总成本、总收入
- 利润额
- 利润率（百分比）
- ROI（百分比）
- 盈亏平衡点

**计算公式**：
- 利润率 = (销售价 - 进货价 - 物流费) / 进货价 × 100%
- ROI = 净利润 / 投资总额 × 100%
- 盈亏平衡点 = 总成本 / (销售价 - 进货价 - 物流费)

**使用场景**：
- 评估单个产品的盈利能力
- 批量采购的成本收益分析
- 定价策略决策

**示例**：
```
进货价：20元
销售价：59元
物流费：5元
销售数量：100件

结果：
总成本：2500元
总收入：5900元
总利润：3400元
利润率：136%
ROI：136%
盈亏平衡点：74件
```

### 5. competitor_analysis_tool - 竞品分析

**功能**：分析指定品类在特定平台上的竞争情况

**参数**：
- `category`（必填）：产品品类，如"面膜"、"手机壳"
- `platform`（可选）：目标平台，默认"淘宝"，可选"拼多多"、"京东"等

**返回结果**：
- 竞品数量
- 市场饱和度（高/中/低）
- 竞品列表
- AI生成的市场分析

**使用场景**：
- 评估市场进入难度
- 了解竞争格局
- 寻找蓝海机会

---

### 6. trend_analysis_tool - 趋势分析

**功能**：分析指定品类的市场趋势和热销关键词

**参数**：
- `category`（必填）：产品品类
- `time_range`（可选）：时间范围，如"1w"（1周）、"1m"（1个月）、"3m"（3个月），默认"1m"

**返回结果**：
- 趋势总结
- 热门品类
- 增长率数据
- 时间范围内的市场变化

**使用场景**：
- 识别增长机会
- 了解季节性趋势
- 把握市场时机

---

### 7. supplier_evaluation_tool - 供应商评估

**功能**：评估和推荐供应商，基于品类、地区和价格范围

**参数**：
- `category`（必填）：产品品类
- `region`（可选）：地区偏好，如"广州"、"浙江"、"义乌"
- `min_price`（可选）：最低进货价
- `max_price`（可选）：最高进货价

**返回结果**：
- 供应商列表
- 供应商资质信息
- 价格区间
- 评估总结

**使用场景**：
- 寻找优质供应商
- 筛选地区性供应商
- 按价格范围筛选

## 🔄 工作流程

智能体遵循以下标准工作流程：

### 1. 需求解析
- 识别用户的核心参数：品类、价格范围、目标平台、地区偏好、起批量等
- 如果用户未提供关键参数，根据常识设置合理默认值

### 2. 市场趋势分析
- 使用 `trend_analysis_tool` 查询相关品类的热销趋势
- 识别热门品类、增长率和潜在机会
- 使用 `web_search_tool` 补充实时市场信息

### 3. 货源搜索
- 使用 `supplier_evaluation_tool` 搜索供应商平台
- 使用 `advanced_search_tool` 筛选特定站点和时间范围的结果
- 从搜索结果中提取：产品名称、供应商、价格、最小起批量、评价等信息

### 4. 视觉参考（可选）
- 如用户需要产品图片参考，使用 `image_search_tool` 搜索产品图片
- 提供图片URL和尺寸信息

### 5. 产品潜力评估
- 使用 `roi_calculator_tool` 精确计算利润率
- 使用 `competitor_analysis_tool` 评估市场饱和度
- 搜索用户反馈和投诉，评估质量风险和知识产权问题
- 综合以上因素给出1-10分的潜力分数

### 6. 推荐生成
- 筛选出Top 5-10个高潜力货源
- 使用Markdown表格呈现推荐结果
- 提供具体的行动建议

### 7. 迭代优化
- 根据用户反馈重新搜索和筛选
- 动态调整搜索条件和过滤标准

## 📤 输出格式

智能体使用以下标准结构输出结果：

### 推荐货源摘要
简要概述找到的货源数量、平均利润率和核心推荐点。

### 详细推荐列表
使用Markdown表格呈现：

| 产品 | 供应商 | 进货价(元) | 预估售价(元) | 利润率 | ROI | 潜力分数 | 推荐理由 |
|------|--------|------------|-------------|--------|-----|----------|----------|
| ... | ... | ... | ... | ... | ... | ... | ... |

### 市场趋势洞察
- 当前热门品类及增长率
- 季节性趋势和机会点
- 竞争态势分析

### 产品图片参考（如适用）
如需要，展示产品图片链接和简要说明

### 行动建议
- 优先测试的供应商
- 建议的销售策略
- 风险提示和注意事项

## 💡 使用示例

### 示例1：基础货源搜索

**用户输入**：
```
帮我找美容护肤类的热卖货源，预算在50元以内，目标平台是淘宝。
```

**智能体会执行**：
1. 解析需求：品类=美容护肤，价格范围=50元以内，平台=淘宝
2. 使用 `trend_analysis_tool` 分析美容护肤市场趋势
3. 使用 `supplier_evaluation_tool` 搜索淘宝平台的美容护肤供应商
4. 使用 `roi_calculator_tool` 计算推荐产品的利润率
5. 输出结构化推荐列表

---

### 示例2：ROI计算

```
如果进货价20元，销售价59元，物流费5元，销售100件的ROI是多少？
```

**智能体会执行**：
1. 使用 `roi_calculator_tool` 计算ROI

## 🎯 最佳实践

### 1. 提供清晰的需求描述
- 明确指定品类、价格范围、目标平台
- 如有地区偏好，请说明
- 提供具体的数据（如进货价、销售价）以便计算ROI

### 2. 充分利用工具
- 询问市场趋势时，智能体会自动使用趋势分析工具
- 需要计算利润时，直接提供价格数据，智能体会使用ROI计算器
- 想了解竞争情况，明确指定平台和品类

### 3. 迭代优化
- 如果对推荐不满意，可以要求排除某些供应商
- 可以调整价格范围、地区偏好重新搜索
- 可以要求更详细的某个品类分析

### 4. 风险控制
- 优先选择资质齐全、有检测报告的供应商
- 建议先小批量试单，测试市场反应
- 注意知识产权风险，避免侵权

## ⚠️ 注意事项

### 搜索限制
- 联网搜索可能受到调用限制，建议合理使用
- 搜索结果可能不包含实时数据，智能体会基于训练数据提供分析

### 数据准确性
- 价格数据基于搜索结果和市场分析，实际价格可能有所波动
- ROI计算基于用户提供的输入，请确保数据准确
- 供应商信息需要进一步核实（如资质、质检报告等）

### 法律合规
- 确保产品符合相关法律法规要求
- 注意知识产权，避免使用侵权产品
- 索要产品检测报告和生产许可证
