# 📱 电商货源猎手 Web 服务 - 快速开始

## ✅ 已完成

你的电商货源猎手智能体现在已经可以通过Web服务在微信中使用了！

## 🚀 快速启动

### 方法1: 使用启动脚本（推荐）

```bash
./scripts/start_web.sh
```

### 方法2: 手动启动

```bash
cd src/web
python3 app.py
```

## 📱 在微信中使用

### 步骤1: 启动服务

运行启动脚本后，你会看到：

```
访问地址:
   - 本地: http://localhost:5000
   - 局域网: http://192.168.1.100:5000  # 你的局域网IP
```

### 步骤2: 在微信中打开

1. 复制**局域网地址**（如 `http://192.168.1.100:5000`）
2. 在微信中发送给"文件传输助手"
3. 点击链接即可打开

### 步骤3: 开始使用

- **文本对话**: 直接输入问题
- **图片分析**: 点击📷按钮上传产品图片
- **流式响应**: AI回复实时显示

## ⚠️ 重要提示

### 网络要求

**方法1: 局域网访问（同一WiFi）**
- 确保电脑和手机连接同一WiFi
- 使用局域网IP地址访问

**方法2: 公网访问（需要配置）**
如果需要让其他网络下的用户也能访问，需要使用内网穿透工具：

推荐使用 **ngrok**:
```bash
# 安装 ngrok
# 访问: https://ngrok.com/download

# 启动 ngrok
ngrok http 5000

# 复制生成的HTTPS链接，在微信中打开
```

## 🛠️ 故障排查

### 问题1: 无法在微信中打开

**解决方案**:
1. 确保电脑和手机在同一WiFi
2. 检查防火墙设置
3. 尝试使用内网穿透工具（ngrok）

### 问题2: 图片上传失败

**解决方案**:
- 确保图片小于16MB
- 使用支持的格式（JPG、PNG、GIF、WEBP）

### 问题3: 连接断开

**解决方案**:
1. 刷新页面重新连接
2. 检查服务是否正常运行
3. 查看服务端日志
